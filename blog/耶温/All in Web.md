All in Web 是我的一个个人计划，目标是：**减少下载各种 App，直接使用 Web 工具完成日常工作和生活需求**。

在这个理念下，我希望将常用的桌面应用功能迁移到 Web 端。无需安装、即开即用。

## Office App

📄 **在线文档编辑器**

访问：https://office.yuwb.dev/

支持 DOCX、DOC、XLSX、XLS、PPTX、PPT、PDF 等主流文档格式的查看和编辑，支持拖拽上传和模板系统。

## Board App

🎨 **在线白板协作工具**

访问：https://board.yuwb.dev/

提供无限画布和丰富的绘图工具，支持多人实时协作，适合团队头脑风暴、流程图绘制等场景。

## Sticky App

📝 **在线桌面便签**

访问：https://sticky.yuwb.dev/

桌面便利贴工具，支持创建多个便签窗口，可自由拖拽、调整大小。支持本地存储，方便记录临时事项。
