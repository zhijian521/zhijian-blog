## 前言

本文介绍如何使用腾讯云快速搭建个人网站。选择腾讯云的原因：

- 域名和服务器统一管理，操作方便
- 轻量应用服务器价格便宜（活动价约 88 元/年）
- 自带宝塔面板，适合新手

当然，你也可以选择其他云服务商，建站流程基本一致。

## 准备工作

开始前需要准备：

- 腾讯云账号
- 已实名认证（购买域名和服务器需要）
- 预算：域名 + 服务器约 100-200 元/年

## 第一步：购买域名

### 搜索域名

访问：https://buy.cloud.tencent.com/domain

输入想要的域名进行搜索。

### 实名认证

新域名需要实名认证才能使用。如果没有信息模板，需要先创建。

选择域名、实名信息模板，提交订单付款即可。

**域名后缀建议：**

- `.cn` - 国内首选，价格便宜
- `.com` - 国际通用，稍贵
- `.top`、`.xyz` - 预算有限可选

## 第二步：购买云服务器

### 选择服务器

推荐使用轻量应用服务器，个人网站足够。

**配置建议：**

- CPU：2 核
- 内存：2 GB
- 带宽：4 Mbps
- 系统：宝塔 Linux 面板

### 获取服务器 IP

购买后记录公网 IP 地址。

路径：控制台 → 轻量应用服务器 → 服务器列表 → 查看 IP

## 第三步：域名解析

将域名指向服务器 IP。

路径：控制台 → 我的资源 → 域名注册 → 选择域名 → 解析

添加记录，记录值填写服务器 IP。

**添加以下记录：**

- 主机记录：`@`，记录类型：`A`，记录值：服务器 IP
- 主机记录：`www`，记录类型：`A`，记录值：服务器 IP

## 第四步：配置服务器

### 配置防火墙

路径：控制台 → 轻量应用服务器 → 防火墙 → 添加规则

添加 8888 端口（宝塔面板默认端口）。

### 获取宝塔账号密码

点击"登录"按钮进入服务器，执行命令：

```bash
sudo /etc/init.d/bt default
```

保存输出的用户名和密码。

### 登录宝塔面板

浏览器访问：

```
http://服务器IP:8888/tencentcloud
```

或

```
http://域名:8888/tencentcloud
```

### 安装环境软件

推荐安装：

- Nginx（Web 服务器）
- MySQL（数据库，如需要）
- PHP（如需要）

### 绑定账号

1. 绑定个人宝塔账号：宝塔面板 → 首页 → 绑定账号
2. 关联腾讯云：宝塔面板 → 首页 → 关联腾讯云

## 第五步：部署网站

### 添加站点

路径：宝塔面板 → 网站 → 添加站点

填写域名，选择 PHP 项目（静态网站也可选）。

### 上传网站文件

点击站点根目录进入文件管理，上传网站文件到根目录。

完成后即可通过域名访问网站。

## 第六步：安全优化

### 必改配置

为了安全，登录后立即修改：

1. **面板端口**（防火墙同步放行新端口）
2. **默认用户名和密码**
3. **安全入口**

路径：宝塔面板 → 面板设置 → 全部设置

### 开启 SSL

建议开启面板 SSL，使用 HTTPS 访问。

## 常见问题

### 域名解析不生效

DNS 解析需要 10 分钟到 48 小时生效，通常几分钟即可。

### 网站无法访问

检查：

1. 域名解析是否正确
2. 防火墙是否开放 80、443 端口
3. 宝塔面板站点是否正常运行
4. 服务器安全组是否放行相关端口

### 宝塔面板无法访问

检查：

1. 防火墙是否开放 8888 端口
2. 服务器是否正常运行
3. 是否修改了默认端口
