# GitHub 中文汉化

> 汉化 GitHub 网站，使中文用户能够更方便地使用 GitHub。


## 涉及工具

-   油猴插件：[Tampermonkey](https://www.tampermonkey.net/)
-   中文汉化脚本：[GitHub 中文汉化脚本](https://github.com/maboloshi/github-chinese)

## 设置步骤

1. Chrome应用商店下载拓展插件：[Tampermonkey](https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)。如果该链接失效，可以直接在拓展商店搜索 `篡改猴`，其他浏览器同理。

2. 安装中文汉化脚本：[GitHub 中文汉化脚本](https://github.com/maboloshi/github-chinese)。访问中文汉化脚本 Github 主页。如下，点击应安装源直接跳转安装：

![安装中文汉化脚本](/images/Imgae_1763364053954.png)

3. 打开开发者模式：浏览器 -> 拓展管理 -> 打开开发者模式

![开发者模式](/images/Imgae_1763364448832.png)

4. 允许运行用户脚本权限：浏览器 -> 拓展管理 -> 篡改猴详情按钮 -> 勾选允许运行用户脚

![脚本权限](/images/Imgae_1763364596486.png)
